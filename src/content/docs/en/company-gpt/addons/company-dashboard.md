With the companyDASHBOARD, you understand how your AI is really being used. The Admin Dashboard gives you a complete overview of CompanyGPT usage in your company – from active users to token consumption to the performance of individual agents.

## KPIs at a glance

The dashboard provides ten meaningful metrics for data-based decisions. At the system-wide level, you can see the total number of configured agents, all registered users, and the active users in the selected time period, each with a trend comparison to the previous period. Inquiries and conversations are also shown by time period.

The core of cost control is the token metrics: Input and output tokens are recorded separately and form the basis for any cost calculation. The picture is complemented by the use of external resources – MCP tool calls, web searches by AI agents, and processed files such as PDFs and documents.

! [Dashboard KPI Overview](admin-dashboard.png)

## AI Agent Statistics

The agent table shows granularly which AI agents are used and how intensively. It can be sorted by requests or token consumption and makes it visible at a glance which model is behind each agent – whether GPT, Gemini or Claude. Input and output tokens are shown separately per agent. Individual agents can be expanded via a drill-down for deeper insights; a search function allows direct access to specific agents or models.

## Time Period Filter & CSV Export

All metrics can be evaluated for any period of time – from individual days to annual overviews. The results can be processed directly in Excel via CSV export or used for internal reports.

## Why companyDASHBOARD?

The companyDASHBOARD creates full transparency: No black box, no blind spots – you can see exactly how your AI is being used. Token and infrastructure costs are always in view, so you can react early to unexpected cost drivers.

For compliance and internal audits, the dashboard logs all agent activity in full and GDPR-compliant. At the same time, it provides the usage figures and productivity data that you can use to concretely prove the ROI of your AI investment.

In addition, you can see which teams and users are actively using AI – and where there is still untapped potential. Combined with agent performance analysis, model selection can be optimized in a targeted manner to get maximum efficiency out of your AI infrastructure.
