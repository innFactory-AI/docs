```markdown
# Copy Editor

## When to use

- User wants to review or improve existing marketing copy
- User asks "Is this copy good?" or "What can I improve?"
- User has a finished draft and wants feedback

## The Seven Sweeps Framework

Review copy in this order:

1. **Clarity** — Does the reader immediately understand what's being said?
   - Simplify overly complex sentences
   - Explain or replace jargon
   - Active voice over passive

2. **Voice & Tone** — Is the copy internally consistent?
   - Does it address the same person throughout?
   - Is the tone appropriate for the brand?

3. **So What?** — Does every statement answer "What's in it for me?"
   - Translate features → benefits
   - Make abstract claims concrete

4. **Prove It** — Are claims backed up?
   - Add numbers, quotes, testimonials
   - Challenge vague superlatives ("the best")

5. **Specificity** — Is the copy concrete enough?
   - "Much faster" → "3× faster"
   - "Many customers" → "500+ companies"

6. **Heightened Emotion** — Does the copy make the reader feel something?
   - Address pain or desire
   - Add story elements

7. **Zero Risk** — Have all barriers to action been removed?
   - Objections addressed?
   - CTA clear and low-friction?

## Common Problems & Fixes

| Problem | Fix |
|---------|-----|
| Passive voice | Rewrite in active voice |
| Sentence too long | Split it, max 20 words |
| No CTA | Add a clear call to action |
| Empty superlatives | Replace with numbers or proof |
| I/we-focused | Rewrite with you/your focus |

## Output Format

1. **Revised copy** — Complete rewrite with all improvements applied
2. **Change list** — What was changed and why
3. **Open questions** — What still needs clarification or research
```
